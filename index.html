Introduction

The computers we build and program are growing up. It's time to make sure that in addition to being so powerful and sophisticated, that they know the difference between right and wrong.

This is about bringing together techniques for how to write software that will seek to do good and will do no harm. These methods, code examples and technologies must be either invented from scratch or found from the public domain. They must be made available and open so that everybody is capable and encouraged to write software that is good. We seek to build an Open Moral Agent.
